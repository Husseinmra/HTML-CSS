EXPLICAÇÕES

Esse tipo de grupo, são tags do corpo do meu site em que eu consigo dar Semântica a um grupo de tags, ou seja, eu consigo dar significado a um grupo de tags.
Dessa forma, eu deixo de usar somente as Divs com IDs e passo a usar tags que já existem no HTML5, como por exemplo:

  • header,
  • footer,
  • section,
  • article,
  • aside,
  • main,
  • nav,
entre outras onde cada uma tem uma função SEMÂNTICAno site.
Antes, usava-se Divs com class e IDs para dar semântica ao site, mas isso não é o correto, pois o HTML5 já possui tags que fazem isso.

Posso usar entao o box shadow e border-radius para dar um efeito de sombra e arredondar as bordas do meu site. Para facilitar eu posso criar este parametro em css e alterar no dev tools para saber exatamente os valores que quero.
Posso tambem usar imagens nas bordas definindo uma borda padrão transparente e em seguida uma border image e então definir se quero que repita ou alongue.
