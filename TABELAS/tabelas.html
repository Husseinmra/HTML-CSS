Tudo sobre Tabelas

Elementos da tabela

Então, quando falo em criação de tabelas, eu tenho os seguintes elementos:

  • table
  • tr
  • th
  • td

O elemento table é o elemento pai, ou seja, o elemento que vai conter todos os outros elementos.
O elemento tr (rows) é o elemento que vai conter as linhas da tabela.
O elemento th (head) é o elemento que vai conter os cabeçalhos da tabela.
O elemento td (data) é o elemento que vai conter os dados da tabela.
Preciso obedecer a hierarquia, onde um elemento fica dentro do outro.

criando linhas

Para criar as linhas da tabela, eu faço um estilo no TD e crio um border, formando uma caixa em volta de cada conteudo de cada Table Data.
Em seguida, se eu quiser que as linhas estejam coladas eu uso o border-collapse: collapse;
E se eu quiser que as linhas estejam separadas, eu uso o border-collapse: separate;

Alinhando conteúdo

Posso usar classes para estilizar as tabelas.
Posso usar o text allign para alinhar o texto dentro do elemento.
Posso usar o vertical allign para alinhar o texto dentro do elemento.

Estrutura de tabelas maiores

Para estruturar uma tabela maior eu posso usar o THEAD, TFOOT e TBODY.
O THEAD é o cabeçalho da tabela, onde eu coloco os títulos das colunas.
O TFOOT é o rodapé da tabela, onde eu coloco os totais.
O TBODY é o corpo da tabela, onde eu coloco os dados.
Eu posso ter dentro de cada um desses elementos, o TR, TH e TD.
O THEAD e o TFOOT são opcionais, mas o TBODY é obrigatório.
Lembrando que tudo deve estar dentro da tag TABLE.

Posso usar tambem o CAPTION para dar um título a minha tabela

Scope e semântica

Semanticamente, para dizer que um conteúdo faz parte de uma determinada linha atrelada a um título eu uso o SCOPE.
Então, no caso da TABELA 2, eu tenho ESTADO como um titulo da COLUNA de todos os 27 estados do Brasil. Nesse caso, eu uso então SCOPE: COL no ESTADO.
No título POPULAÇÃO os conteúdos também estão em uma coluna abaixo dele então eu uso SCOPE: COL.
Já no FOOT eu tenho que TOTAL DE HABITANTES é um número que se encontra ao seu lado, ou seja, na mesma linha, logo eu uso o SCOPE: ROW.
O SCOPE é um atributo que eu coloco dentro de títulos (TH) e seus resultados, e ele pode ser COL ou ROW.

Se meu titulo tiver sobre ele ou ao seu lado, mais de uma linha eu uso scope: colgroup e rowgroup respectivamente

Importante ressaltar que o scope esta ligado aos RESULTADOS daquela palavra e nao somente a palavra em si.

Mesclagem de celulas

Para utilizar o SPAN, propriedade que mescla duas ou mais células, eu uso o atributo COLSPAN e ROWSPAN.
O COLSPAN mescla as colunas e o ROWSPAN mescla as linhas.
Exemplo na tabela 3: eu apaguei a letra C e defini que a letra B ocuparia seu espaço, ou seja, o espaço de duas colunas. Logo: COLSPAN=2.
Apaguei tambem a letra G, defini que a letra D ocupasse seu lugar, logo ela ocuparia o espaço de duas linhas, então ROWSPAN=2.

Se eu quiser alterar o tamho de apenas uma coluna na minha tabela, eu uso o style="width: 90px;" dentro da tag TH ou no TD.

Destacando colunas

Se eu quiser destacar uma coluna inteira eu poderia definir o mesmo class para todos os elementos dessa coluna, porém se ela for muito grande eu posso fazer isso de uma vez só usando o COLGROUP.
Para utilizá-lo eu abro a tag coulgroup e dentro dela coloco a tag col, sendo que cada col que eu colocar dentro do colgroup vai representar uma coluna da minha tabela respectivamente.
Devo me lembrar de atribuir ao título da tabela (th) o Scope col para que formate todos os elementos embaixo dele. Dentro de cada Col eu defino um Class e no meu style personalizo esse Class.
