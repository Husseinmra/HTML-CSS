Tudo sobre Tabelas

Então, quando falo em criação de tabelas, eu tenho os seguintes elementos:

  • table
  • tr
  • th
  • td

O elemento table é o elemento pai, ou seja, o elemento que vai conter todos os outros elementos.
O elemento tr (rows) é o elemento que vai conter as linhas da tabela.
O elemento th (head) é o elemento que vai conter os cabeçalhos da tabela.
O elemento td (data) é o elemento que vai conter os dados da tabela.
Preciso obedecer a hierarquia, onde um elemento fica dentro do outro.

Para criar as linhas da tabela, eu faço um estilo no TD e crio um border, formando uma caixa em volta de cada conteudo de cada Table Data.
Em seguida, se eu quiser que as linhas estejam coladas eu uso o border-collapse: collapse;
E se eu quiser que as linhas estejam separadas, eu uso o border-collapse: separate;

Posso usar classes para estilizar as tabelas.
Posso usar o text allign para alinhar o texto dentro do elemento.
Posso usar o vertical allign para alinhar o texto dentro do elemento.
