METODO INLINE

Aqui utilizamos o metodo externo, atraves de um LINK no head do documento. Desta forma, teremos uma aba separada somente para estilo em css sozinho (standalone). A vantagem é que tudo que eu alterar lá, valerá para todos os docuemntos que eu tiver o link ativo. Ainda, posso usar os 3 estilos: Inline, local e externo, porem devo evitar ao máximo o inline por ser mais trabalhoso, Neste arquivo eu usei os 3 para exemplificar. O metodo inline tem preferência sobre os demais.
