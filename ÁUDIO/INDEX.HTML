ABRINDO MÍDIAS DE DIFERENTES TAMANHOS

Tente abrir estas mídias em diferentes dispositivos

Então, quando eu quiser carregar multiplas imagens de acordo com a largura do dispositivo, eu crio a tag picture e defino um MAX WIDTH (mudar a tag original) e defino os PIXELS de forma hierarquica. Do menor para o maior. Devo antes carregar uma imagem que eu quero que abra usando a tag img normalmente (usar a maior versão).

ÁUDIOS

Vamos tentar reproduzir áudios

Então, eu crio a tag audio e dentro dela posso colocar parâmetros como preload e definir se o site so irá carregar quando carregar todo o site - auto - (evitar), ou carregar infos básicas com o metadata e none que só irá carregar quando o leitor clicar para reproduizr. O controls é para que o usuário possa controlar a reprodução.Posso usar o auto play e o loop também. Se o navegador não conseguir reproduzir, eu posso criar um link para o arquivo.

Lembrando que eu posso usar o mp3 (type será mpeg), posso usar o ogg (type será ogg) e posso usar o WAV (evitar por ser muito pesado)

Posso tambem criar outras linhas de source src em diferentes formatos, de forma hierarquica, para que o site tente carregar o mesmo arquivo em diferentes formatos sucessivamente.
