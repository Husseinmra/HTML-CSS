Usando Links externos (âncoras)

Acessando o meu perfil no YOUTUBE

Para eu criar um link que abra uma nova guia, AO INVÉS DE SOBREPOR O MEU SITE NA ABA, eu devo usar TARGET= _BLANK E UM REL= "EXTERNAL" após a referencia da tag

Posso criar links dentro do meu próprio site.

Posso referênciar um novo link interno dentro do meu site. Ele levará á uma nova aba que terá conteúdos meus.

Evite, em caso de links internos, usar o _blank, pois o usuário abrirá muitas abas pra navegar dentro do meu próprio site. Utilizar sempre para conteúdos externos, para que ele nao perca meu site nas abas.

Desta forma, eu escolho a palavra ou frase que eu quero transformar em link e envelopo ela com a tag <A>

Você tambem poderá acessar nossa PAGINA de receitas para te ajudar nesta jornada.

Quando eu criar uma pasta dentro do meu repositório, tenho que fazer o caminho da pasta para referenciar ela

Se nesta nova página eu quiser voltar para a primeira página, que esteja fora desta pasta, para encontrá-la eu uso o caminho ../ dentro das aspas na TAG, pois quando eu usar o CTRL e espaço ele não irá encontrar itens fora da pasta em que esta página estiver.
