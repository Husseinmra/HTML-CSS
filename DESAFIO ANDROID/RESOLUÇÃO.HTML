o Professor ensinou sobre uma maneira global de mexer no CSS através do :root e dentro dele definir as variaveis com (--), assim posso usar essas variaveis em qualquer lugar do CSS, por exemplo: :root { --cor1: #000000; --cor2: #FFFFFF; --cor3: #FF0000; --cor4: #00FF00; }
Então, quando eu for definir uma cor para o meu título, por exemplo, eu posso usar a variavel assim:
h1 { color: var(--cor1); }
Essa cor estará disponível no final da lista de cores também.
