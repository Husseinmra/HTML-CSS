Criação de Listas

Listas ordenadas

  1. acordar
  2. lavar roupa
  3. escovar os dentes
  4. ligar o computador
  5. Ir para o trabalho
  6. tomar banho
  7. treinar
  8. dormir

Então, quando quero criar listas ordenadas, uso a tag OL de ordered list e posso definir parâmetros dela usando o Type, dizendo que tipo de marcação quero na lista e o start para saber onde ela ir iniciar.

Listas não ordenadas

  • acordar
  • lavar roupa
  • escovar os dentes
  • ligar o computador
  • Ir para o trabalho
  • tomar banho
  • treinar
  • dormir

Então, neste tipo de lista nao existe numeração, podendo ser criada envelopada pela tag de UL (unordered list)

Nesta tag, posso inserir o atributo de type para dizer qual tipo de marcação quero na lista, podendo ser disc, circle ou square.

    Entao, posso juntar listas ordenadas e nao ordenadas, criando uma lista dentro de outra lista, e assim por diante.
    Também posso criar listas de definição, onde tenho um termo e sua definição, usando a tag DL de definition list, e dentro dela, uso a tag DT de definition term e a tag DD de definition description.
