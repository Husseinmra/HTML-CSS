Tudo sobre formulários

Formulários são uma parte essencial da web, permitindo que os usuários interajam com sites e aplicativos. Eles são usados para coletar informações, como nomes, endereços, emails e preferências.

Para criar um formulário eu devo usar a tag FORM edentro dela definir o que vai ser enviado para o servidor.
Desta forma eu crio um nome e sobre nome e um botão de enviar por exemplo.
Para criar a "caixinha" de texto eu devo usar a tag INPUT e nela definir o tipo de input que vai ser criado.
No caso de input de texto eu devo usar o input text e para criar o botão de enviar eu devo usar o input submit.

O input tem elementos como name e id que são usados para identificar o input.
Posso colocar nomes iguais para o name e id, mas não é uma boa prática.
O name é o nome que vai ser enviado para o servidor e o id é o id do input.

Quando eu crio um formulário, além de ter botões e caixinhas precisa haver semântica entre o input e o nome que está escrito.
Para isso eu devo usar a tag LABEL no texto fora da caixa e dentro dela colocar o id do input.
Desta forma, quando o usuário clicar no nome, a caixinha de texto vai ser selecionada.

Existe também os métodos GET e POST que são usados para enviar os dados para o servidor.
O GET é usado para enviar dados sem segurança e o POST é usado para enviar dados com segurança e arquivos maiores (com mais de 3000 caracteres).

outros atributos que posso definir no input são:
placeholder: que é o texto que vai ser exibido dentro da caixinha de texto. Como exemplo: "Digite seu nome".
required: que é um input que é obrigatório ser preenchido. Se não for preenchido, o formulário não vai ser enviado e exibirá uma mensagem de erro automaticamente.
minlength e maxlength: que são o mínimo e máximo de caracteres que podem ser escritos na caixinha de texto.
size: que é o tamanho da caixinha de texto relacionada ao número de caracteres, lembrando que isso não é a quantidade de caracteres que pode ser escrito, mas sim o tamanho que será exibido.
autocomplete: que é um parâmetro utilizado no form e no input para definir se o navegador vai preencher automaticamente o input com os dados já escritos pelo usuário. Se eu habilitar isso no form eu posso definir o tipo de autocomplete no input como senha, usuario, idade e etc

Posso também criar os botões de envio e limpar o formulário usando o input do tipo submit e do tipo reset respectivamente.

Alguns formatos de input mais conhecidos são:

  • text
  • password
  • email
  • number
  • date
  • tel
  • color
  • range
  • search
  • url
  • week
  • month
e em cada um deles posso posso definir o VALUE que é um valor padrão que vai ser exibido na caixinha de texto e que tenha relação com aquele dado, diferente do placeholder em que eu posso colocar o que eu quiser.
