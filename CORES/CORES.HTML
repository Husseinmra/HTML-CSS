OLHAR DESCRIÇÕES E EXPLICAÇÕES DENTRO DO CÓDIGO

HARMONIZAÇÃO DE CORES

As cores podem ser harmonizadas de acordo com o círculo cromático, que é uma representação gráfica das cores e suas relações.

CORES PRIMÁRIS

As cores primárias são aquelas que não podem ser formadas pela mistura de outras cores. Elas são a base para a criação de todas as outras cores.

  • Vermelho
  • Azul
  • Amarelo

CORES SECUNDÁRIAS

  • ROXO
  • LARANJA
  • VERDE

CORES TERCIÁRIAS

  • AMARELO-ESVERDEADO
  • AMARELO-ALARANJADO
  • VERMELHO-ARROXEADO

Cores terciárias sao cores misturadas e pasteis

TEMPERATURA DAS CORES

Estão a esquerda do círculo cromático as cores frias e à direita estão as cores quentes

cores complementares

São cores que constrastam entre sí. Para encontrá-las é so traças uma linha e achar a cor oposta.

as cores análogas nao possuem o contrastes. São cores vizinhas.
