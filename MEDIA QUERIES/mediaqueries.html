Tudo sobre media queries

O que são?

As media queries nada mais são do que uma forma de criar estilos diferentes para diferentes dispositivos.
Eles são arquivos css separados que são chamados de acordo com a necessidade do dispositivo. Para isso, dentro da tag do CSS eu adiciono a tag media= e o tipo de dispositivo que eu quero que o estilo seja aplicado.
Exemplos de media queries:

  • media="screen" - para telas
  • media="print" - para impressão
  • media="all" - para todos os dispositivos

Media features

As media features são uma forma de criar estilos diferentes para diferentes dispositivos.
Elas são escritar entre parenteses em seguida do media type
Exemplo:
  • media= "screen and (max-width: 768px)" - para telas menores que 768px
  • media= "screen and (orientation: portrait)" - para telas em modo retrato
  • media= "screen and (orientation: landscape)" - para telas em modo paisagem

Usando o @media

O @media é uma forma de criar estilos diferentes para diferentes dispositivos de maneira unificada.
Ele é escrito no style local de CSS ou em um unico arquivo CSS para unificar todos os estilos em um só arquivo.
Exemplo:

@media screen and (max-width: 768px) { body { background-color: red; } } @media screen and (orientation: portrait) { body { background-color: red; } }

@media screen and (max-width: 768px) { body { background-color: red; }

Então eu posso ter Varios arquivos de CSS com media types diferentes com seus respectivos media features no HEAD para cada dispositivo (recomendado), posso abrir um STYLE local no meu HTML e escrever o @media para que cada um tenha uma configuração diferente para cada dispositivo que eu quero que o estilo seja aplicado, ou posso abrir um unico arquivo CSS e escrever o @media para cada dispositivo que eu quero que o estilo seja aplicado.

Devices Breakpoints

Os devices breakpoints são os pontos de quebra de tela que eu quero que o estilo seja aplicado.
Eles são escritos em pixels e são usados para criar estilos diferentes para diferentes dispositivos.
Exemplo:

  • 1200px + : Telas Grandes
  • 992px - 1200px: Desktop
  • 768px - 992px: Tablet
  • 600px - 768px: Mobile
  • Até 600px: Telas Pequenas

Para utilizar os devices breakpoints, eu posso usar o @media para criar estilos diferentes para cada dispositivo.
Exemplo:

  • @media screen (min-width: 992px) and (max-width: 1200px) {}

Neste caso, o estilo será aplicado para todos os dispositivos que estão entre 992px e 1200px. Caso eu queira que uma imagem apareça apenas nessa media eu dou display none para os outros que ja carreguei e coloco display block para a que eu quero que apareça.
