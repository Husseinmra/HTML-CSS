METODO INLINE

O metodo local é vantajoso para quando quisermos adicionar parâmetros de estilo para vários pontos do conteudo de uma vez só. Para isso é só usar a tag que quero modificar, abrir o colchete e escrever as tags de estilo em CSS. A parte ruim é que ficará muito extenso em estilo até chegar no conteúdo e sempre que eu quiser ter varias páginas, terei que ficar copiando todos os parâmetros do HEAD para as outras páginas e ficar alterando sempre que meu cliente pedir.
