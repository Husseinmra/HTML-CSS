TESTANDO ÍCONE PARA SITE

Para criar esta pequena imagem da ABA, deve usar arquivo com extensão ICO (icon), utilizando sites como ICON ARCHIEVE, onde haverá diversos icones prontos.

Para criar um novo ícone posso utilizar o FAVICON.CC ou o FAVICON.IO

Após a criação, colocar o arquivo na pasta do site, e na aba HEAD, adicionar "link favicon" e adicionar a imagem desejada.
